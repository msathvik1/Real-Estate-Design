# Real-Estate-Design

# Overview
Designed Apna Makaan, a real estate application that helps users buy and rent properties. Features include property listings, market trends, real estate articles, and verified reviews. Focused on creating a clean, intuitive, and user-friendly UI/UX using Figma.
